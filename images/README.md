# Instrucciones para Agregar las Fotos

Por favor, guarda las siguientes imágenes en esta carpeta (`images/`):

## 1. Foto de Perfil (Fondo Blanco)
- **Archivo:** `profile.jpg`
- **Descripción:** La foto con fondo blanco del jugador con medalla
- **Uso:** Se mostrará como imagen de perfil en la sección hero (círculo)

## 2. Foto de Fondo (Campo de Fútbol)
- **Archivo:** `background.jpg`
- **Descripción:** La foto del jugador en acción en el campo
- **Uso:** Se mostrará como imagen de fondo en la sección hero

## Pasos:
1. Guarda la foto con fondo blanco como `profile.jpg` en esta carpeta
2. Guarda la foto del campo como `background.jpg` en esta carpeta
3. Recarga la página web para ver los cambios

---

Las imágenes ya están configuradas en el código y se cargarán automáticamente cuando las guardes con estos nombres.
